# 使用氧气 {#id221BH0V0UE8}

氧气XML Author具有用户友好的创作功能，并提供易于使用的界面。

您可以使用相同的源生成PDF、ePub、HTML和许多其他格式的输出。

*请联系您的客户成功团队以在环境中启用此功能。 这并非开箱即用支持的一部分。*
