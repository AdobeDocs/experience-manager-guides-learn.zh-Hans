# 翻译内容 {#id1645H700L67}

AEM指南附带强大的功能，使您能够将内容翻译成多种语言。 AEM指南支持人文和机器翻译工作流。

- **人文翻译**  — 资产将发送给您的翻译提供商，并由专业翻译人员翻译。 完成后，将返回已翻译的资产，然后将其导入回AEM。

- **机器翻译**  — 这是AEM中的默认服务，其中内容使用机器翻译实时立即翻译。 现成的AEM提供连接到Microsoft Translator的功能。

   >[!NOTE]
   >
   > Microsoft Translator仅作为试用许可证提供。
