# 了解输出预设 {#understanding_output_settings}

AEM指南支持以PDF、HTML5和EPUB格式为FrameMaker文档创建输出。 使用这些输出格式，您可以配置各种输出预设。 输出预设表示您希望在其中发布内容的自定义输出格式。

以下各节将介绍支持的输出格式可用的选项。
