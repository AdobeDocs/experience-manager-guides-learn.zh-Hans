# 创建和预览主题 {#id21BQE0RJ042}

AEM指南附带一项功能强大的功能，可从Assets UI创建DITA主题。 然后，您可以从资产UI中预览主题。 “预览”模式还提供了许多功能强大的功能，可帮助您执行诸如查看元数据、在Web编辑器中编辑主题、打开源以及查找基于Web的版本差异等关键操作。
