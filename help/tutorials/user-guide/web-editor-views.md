# Web编辑器视图 {#id204GK0D0V5Z}

AEM指南的Web编辑器支持以三种不同模式或视图查看文档：

## 创作

这是Web编辑器的典型视图。 您可以像在任何常规富文本编辑器中一样编辑主题。 在“创作”视图中，您可以选择保存文档的修订版本、查找和替换内容、插入元素、插入超链接、插入内容引用等。

>[!NOTE]
>
> 使用内容引用时，引用的内容还会以蓝色显示在“创作”视图中。 引用的内容不可编辑。

## 源

“源”视图显示构成主题的基础XML。 如果您习惯直接使用XML，则应使用“源”视图。 除了在此视图中进行常规文本编辑之外，您还可以使用智能目录添加元素和属性，或查找并替换文本、元素或属性。

- 要调用智能目录，请将光标放在要插入新元素的任何元素标记的末尾，然后输入“&lt;”。 编辑器显示可在该位置插入的所有有效XML元素的列表。 使用箭头键选择要插入的元素，然后按Enter。 输入结束括号“\>时，将自动添加元素的结束标记。

   ![](images/smart-catalog-elements.png){width="400" align="left"}

- 您还可以从“源”视图轻松更改元素。 例如，如果更改 `p` 元素 `note`，则结束 `p` 标记会自动更改为 `/note`. 如果将元素替换为不正确的元素，则会立即显示验证错误。

- 如果要向元素添加属性，请将光标放在元素标记内，然后按空格键。 该元素的有效属性列表将显示在智能目录中。 使用箭头键选择所需的元素，然后按Enter键插入该元素。 要为属性指定值，请输入等号\(=\)，编辑器会自动输入开号和闭号“”，您可以在其中指定属性的值。

   ![](images/smart-catalog-attribute.png){width="350" align="left"}

- 在“源”视图中，有一个“自动缩进”选项，该选项可以以呈现且易于读取的格式重新组织XML代码。 此外，如果选择任何文本并从“创作”视图切换到“源”视图或从“源”视图切换到“创作”视图，则所选文本也会在其他视图中突出显示。
- “源”视图的另一个强大功能是文档中的XML验证。 如果打开包含无效XML的文档，则会在“源”视图中打开该文档，其中包含有关无效XML的信息。 例如，在以下屏幕截图中，在“解析错误”弹出窗口中提供了有关错误XML的确切信息。

   ![](images/invalid-topic-xml.png){width="650" align="left"}

   在上面的屏幕截图中，使用交叉高亮指示包含错误XML的行。

- “查找和替换”功能允许您在源视图中搜索任何文本、元素或属性。
有关更多详细信息，请参阅 **查找和替换** 中的功能描述 [主工具栏](web-editor-features.md#id#id2051EA0G05Z) 中。

- “源视图”提供了许多快捷键，帮助您快速导航和处理文档。 下表列出了支持的操作及其快捷键：

## 预览

在“预览”模式下打开主题可显示用户在浏览器中查看主题时该主题的显示方式。 对于DITA映射，将显示映射的预览，其中显示映射中所有主题的单个复合文档。

预览模式提供以下功能：

- [根据条件过滤器查看内容](#id2114BI00VXA)
- [查看跟踪更改标记](#id2114BJ00CE8)
- [将主题导出为PDF](#id2114BL00B5U)

### 根据条件过滤器查看内容 {#id2114BI00VXA}

如果您在主题或映射中使用了条件，则这些条件会显示在过滤器面板中。 默认情况下，会选择所有条件并显示整个内容。 如果取消选择某个条件，则具有该条件的内容将从视图中删除。 您还可以选择突出显示条件化内容。

下图显示了一个使用两个条件的主题： `Audience` 和 `Product`. 条件化内容以黄色背景突出显示。

![](images/preview-filters.png){width="800" align="left"}

### 查看跟踪更改标记 {#id2114BJ00CE8}

如果文档包含跟踪更改标记\（或视觉提示\），则您还可以预览包含或不包含这些标记的文档。 在预览文档时，右侧面板包含过滤器和跟踪选项。

![](images/preview-tracking_cs.png){width="400" align="left"}

有三个 **跟踪** 选项：

- **无标记**:在此视图中，所有插入和删除都被接受，并呈现文档的简单视图。 在此视图中，您看不到任何跟踪更改标记。
- **原始**:在此视图中，所有插入都被拒绝，所有删除都被恢复，然后显示预览。 只需在启用跟踪更改模式之前获取文档的原始形式即可。
- **显示标记**:在此视图中，您将获得插入和删除内容的所有标记。

   下图显示了带有标记的映射文件的预览：

   ![](images/preview-map-with-track-changes.PNG){width="800" align="left"}


### 将主题导出为PDF {#id2114BL00B5U}

PDF是在文档开发周期的每个可能阶段使用的最常见的输出格式之一。 AEM指南可让您灵活地生成单个主题或整个映射文件的PDF。 导出为PDF功能允许作者、发布者或管理员轻松地为单个主题生成PDF输出。 它使用在文件夹级别配置文件中保存的DITA-OT配置来生成PDF。

此功能支持以下功能：

- 生成主题当前活动工作副本的PDF。
- 接受DITA-OT转换名称和命令行参数以生成PDF。
- 在本地系统上保存生成的输出。
- 在生成输出之前，解析主题中使用的键和内容引用。

要将主题导出为PDF，请执行以下步骤：

1. 在预览模式下打开主题。

1. 单击 **导出为PDF** \(![](images/export-as-pdf-icon.svg)\)图标。

   将显示导出为PDF对话框。

   ![](images/export-as-pdf-dialog.png){width="350" align="left"}

1. *\（可选\）* 指定DITA-OT转换名称以及要使用的任何命令行参数。

1. 单击&#x200B;**下载**。

   >[!NOTE]
   >
   > 确保您已在浏览器配置中启用弹出窗口，否则将不会下载PDF。

   PDF将在新选项卡中生成并打开，或者您会看到一个对话框，在本地系统上保存PDF。
