# 了解Web编辑器功能 {#id176NC500V5Z}

此部分将指导您逐步了解Web编辑器中提供的各种功能。 我们可以将Web编辑器分为以下部分或区域：

- [主工具栏](#id2051EA0G05Z)
- [辅助工具栏](#id2051EA0J0Y4)
- [左面板](#id2051EA0M0HS)
- [内容编辑区域](#id2051EB000UI)
- [右面板](#id2051EB003YK)

以下小节详细介绍Web编辑器的各个部分。

## 主工具栏 {#id2051EA0G05Z}

主工具栏位于Web编辑器界面的顶部，它提供了Web编辑器中可用的文件级功能和各种创作模式。 顶部工具栏中可用的功能说明如下：

**全部保存** - ![](images/SaveFloppy_icon.svg)

保存您在所有打开的主题中所做的更改。 如果在Web编辑器中打开了多个主题，请单击 **全部保存** 或使用 **Crtl**+**S** 快捷键可一键保存所有文档。 您不必单独保存每个文档。

>[!NOTE]
>
> 保存操作不会创建主题的新版本。 要创建新版本，请选择另存为新版本。

**另存为新版本** - ![](images/save-revision-icon.png)

保存您在主题中所做的更改，并创建主题的新版本。 如果您正在处理新创建的主题，则版本信息将显示为 **无**.

![](images/save-all-first-version-none_cs.png){width="800" align="left"}

版本号会随为主题或映射文件创建的每个新版本而发生更改。

当您选择使用 **另存为新版本**，将出现以下对话框：

![](images/save-as-new-version-dialog.PNG){width="300" align="left"}

输入注释和版本标签以标识更改并单击 **保存** 创建文件的新版本。

当您选择 *另存为新版本*，则会在DAM中创建主题的第一个版本，该版本也将成为您主题的当前活动版本。 稍后，如果您还原到该主题的旧版本，则该版本将成为该主题的当前活动版本。

如果您的管理员已预配置了版本标签，那么您将在下拉列表中看到这些标签。 您可以从可用标签列表中选择标签并保存文档。

![](images/web-editor-pre-defined-labels.PNG){width="300" align="left"}

在保存主题时，您可以添加一个注释，以指定您在主题中所做的更改。 此评论显示在主题的版本历史记录中。

如果您的主题正在审阅中，审阅人将收到通知，指出该主题有较新版本可用。 他们可以轻松访问文档的最新修订版并继续查看主题的最新版本。

将指针悬停在主题标题上时，将显示文件路径和版本号。

![](images/mouse-hover-on-title_cs.png){width="800" align="left"}

>[!NOTE]
>
> 在主题的某个版本可用后，您还可以向主题添加标签。 然后，可以使用这些标签为发布特定版本的文档创建基线。 有关在主题中使用标签的更多信息，请参阅 [使用标签](web-editor-use-label.md#).

**撤消和重做** - ![](images/Undo_icon.svg) / ![](images/Redo_icon.svg)

撤消或重做上一个操作。

**删除元素** - ![](images/Delete_icon.svg)

删除当前选定的元素或放置光标的元素。

**查找和替换** - ![](images/FindAndReplace_icon.svg)

“查找和替换”功能在“创作”和“源”视图模式中可用。 “查找并替换”文本栏显示在主题编辑区域的底部。 您可以使用快捷键 **CTRL**+**F** 调用查找和替换栏。

![](images/find-replace-bar.png){width="800" align="left"}

使用设置图标\(![](images/settings-find-replace-icon.svg)\)时，您可以将 **忽略大小写** 和 **仅全字** 搜索选项。 要执行不区分大小写的搜索，请打开\（或选择\） **忽略大小写** 选项。 否则，如果要执行区分大小写的搜索，请关闭\（或取消选择\） **忽略大小写** 选项。 您还可以选择搜索整个词。

搜索是即时的，这意味着在 **查找** 字段中，将立即搜索并选择该术语。 同样，要替换主题中的文本，请在相应字段中输入搜索词及其替换项，然后单击 **替换** 或 **全部替换** 按钮。

在“源”视图中，“查找和替换”对于搜索特定元素或属性非常有用。 例如，如果要将 `@product` 属性，则可以从“源”视图轻松完成该操作。 “创作”视图不允许您基于属性或元素进行搜索。 但是，在使用 **全部替换** 功能，因为它可能会覆盖XML代码。

**编辑器设置** - ![](images/editor_settings_icon.svg)

编辑器设置仅适用于管理用户。 使用首选项，管理员可以配置以下设置：

>[!NOTE]
>
> 如果要更新任何默认设置，则应重新打开文档以使更改生效。

- **常规**:“常规”设置允许您配置字典以与Web编辑器一起使用。 此选项卡包含三个部分： **拼写检查**, **条件**&#x200B;和 **创作**.

   ![](images/editor-setting-general.png){width="650" align="left"}

   - **拼写检查**:有两种选择 —  **AEM拼写检查** 和 **浏览器拼写检查**. 默认情况下，编辑器使用浏览器拼写检查功能，其中使用浏览器的内部版本词典执行拼写检查。 您可以切换到AEM拼写检查以使用AEM词典，该词典也可进行自定义以添加自定义单词列表。 有关自定义AEM词典的更多信息，请参阅 *自定义AEM默认词典* 安装和配置Adobe Experience Manager指南as a Cloud Service中的部分。


   - **条件**

      - **在创作视图中突出显示条件文本**:选择此选项可在创作视图中突出显示条件文本。 条件内容使用为条件定义的颜色突出显示。

      - **使用条件属性验证**:选择此选项可允许验证为属性定义的值。 这会阻止您添加任何不正确的值。

      - **在主题方案面板中显示带标题的键**:选择此选项可显示主题方案中的键和标题。 如果不选择此选项，则仅显示标题。 例如，此处还显示键“os”、“audience”和“other”以及标题。

         ![](images/subject-scheme-title.png){width="550" align="left"}

      - **在“条件”面板中显示主题方案**:选择此选项可在条件面板中查看主题方案。 如果取消选择此选项，则定义的条件将显示在条件面板中。
   - **创作**

      - **启用全部替换**:选择此选项可查看“查找并替换”面板中的“全部替换”图标。


**面板**:此设置控制编辑器左侧面板中显示的面板。 您可以切换开关以显示或隐藏所需的面板。

![](images/editor-setting-panel.png){width="650" align="left"}

>[!NOTE]
>

- **元素列表**:作为管理员，您可以使用 [插入元素](#id204SG30105Z) ，并定义元素的显示名称。 元素列表设置允许您根据DITA规范指定元素的名称，并指定要用来代替DITA定义的元素名称的标签：

   ![](images/editor-setting-element-list.png){width="650" align="left"}

在以上屏幕截图中， `b` 元素的标签为粗体， `codeblock` 为指定了代码块的标签以及其他一些元素。 如果您选择 **仅使用以上元素** 选项，则“插入元素”弹出窗口中将仅显示此列表中的有效元素\（在当前插入点\）。

在以下屏幕截图中，上一个屏幕截图中的4个已配置元素中，只有3个显示在当前上下文中：

![](images/editor-setting-insert-element-list.PNG){width="300" align="left"}

- **属性列表**:与元素列表类似，您可以控制要在元素的属性列表中显示的属性列表及其显示名称。 在以下屏幕截图中，仅配置了3个属性以在元素的属性列表中显示：

![](images/editor-setting-attributes-list.png){width="650" align="left"}

使用此设置时，当您尝试向元素添加属性时，您只会看到列表中配置的属性列表。

![](images/editor-setting-add-attributes-list.png-to-element.PNG){width="300" align="left"}

- **显示属性**:与“属性列表”一样，您可以控制要在元素的属性列表中显示的属性列表。 默认情况下，四个 **显示属性**  — 已将受众、平台、产品和prop配置为在元素的属性列表中显示。 您还可以使用 **添加** 图标。 您还可以使用 **删除** 图标。

为元素定义的属性显示在“布局”和“大纲”视图中。

![](images/editor-settings-display-attributes.png){width="550" align="left"}

- **翻译**:此选项卡包含将源标签传播到目标版本的选项。

   - **将源版本标签传播到目标版本**:选择此选项可将源文件版本的标签传递到已翻译的文件。 默认情况下，此设置处于禁用状态。

   ![](images/editor-setting-translation.png){width="550" align="left"}


**用户首选项** - ![](images/user_preference_editor_icon.svg)

所有用户首选项均可供所有作者使用。 使用首选项，作者可以配置以下设置：

![](images/user_preference_editor.PNG){width="550" align="left"}

- **主题**:您可以从编辑器的“光”、“最亮”、“深”或“最暗”主题中进行选择。 对于最浅的主题，工具栏和面板使用更浅的灰色背景。 对于“浅色”主题，工具栏和面板使用浅灰色背景。 对于“最暗”主题，工具栏和面板使用更深的黑色背景。 对于深色主题，工具栏和面板使用黑色背景。 在所有主题中，内容编辑区域以白色背景显示。

- **文件夹配置文件**:文件夹配置文件控制与条件属性、创作模板、输出预设和Web编辑器配置相关的各种配置。 默认情况下，会显示全局配置文件。 此外，如果管理员已在系统中配置了文件夹配置文件，则这些文件夹配置文件也会显示在文件夹配置文件列表中。

   Web编辑器的配置，管理员可以在文件夹配置文件中定义这些配置包括：自定义用户界面，包括工具栏图标、Web编辑器布局、代码片段和根映射。 有关更多详细信息，请参阅 *配置全局或文件夹级配置文件* 在安装和配置Adobe Experience Manager指南as a Cloud Service中。

   >[!NOTE]
   >
   > 当前文件夹配置文件的名称将作为主工具栏中用户首选项图标的标签显示。

- **基本路径**:默认情况下，当您从Web编辑器访问AEM存储库时，将从/content/dam位置显示资产。 您的工作文件夹很可能是/content/dam/文件夹内的几个文件夹。 每次需要单击几下才能访问工作文件夹。 您可以设置工作文件夹的基本路径，然后设置“存储库视图”，以前显示该位置的内容。 这可缩短访问工作文件夹的时间。 此外，在主题中插入任何引用或媒体文件时，文件浏览位置以“基本路径”中设置的文件夹开头。

- **选择根映射**:选择DITA映射文件以解析密钥引用或术语表条目。 选定的根映射具有最高的优先权来解析键引用。 有关更多详细信息，请参阅 [解决键引用](map-editor-other-features.md#id176GD01H05Z).


>[!NOTE]
>
> 如果您不想使用任何根映射，请确保 **选择根映射** 字段，此时该字段将保留为空。

**创作、源和预览模式**

有关各种创作和文档查看模式的详细信息，请参阅 [Web编辑器视图](web-editor-views.md#).

## 辅助工具栏 {#id2051EA0J0Y4}

在Web编辑器中打开要编辑的主题时，会显示辅助工具栏。 辅助工具栏中可用的功能说明如下：

**插入元素** - ![](images/Add_icon.svg)

在当前或下一个有效位置插入有效元素。 如果您在块元素中工作，如 `note`，然后使用插入元素图标在 `note` 元素。 在以下屏幕截图中，在p \(paragraph\)元素内插入了注释元素：

![](images/note-in-para-insert-element_cs.png){width="800" align="left"}

如果在备注元素中按Enter，则在备注元素本身中创建新段落。 要在注释外插入新元素，请在元素痕迹导航中单击p元素\（在屏幕截图中突出显示\），然后单击插入元素图标或按 ***Alt***+***输入*** 打开“插入元素”弹出窗口。 然后，选择所需的元素并按Enter键，以在注释元素之后插入所选元素。

当出现闪烁的块光标时，您还可以在两个元素之间添加一个元素。

![](images/Block-cursor.png){width="300" align="left"}

例如，如果您正在处理DITA主题，并且块光标在简短描述和主体之间闪烁，则可以添加 `prolog` 元素，然后添加版权、作者和其他详细信息。

输入新元素的另一种方法是使用上下文菜单。 在文档中的任意位置右键单击以调用上下文菜单。 从此菜单中选择“插入元素”以显示“插入元素”对话框，然后选择要插入的元素。

![](images/insert-element-before-after.png){width="300" align="left"}

**插入段落** - ![](images/Paragraph_icon.svg)

在当前或下一个有效位置插入段落元素。

**插入/删除编号列表** - ![](images/TextNumbered_icon.svg)

在当前或下一个有效位置创建编号列表。 如果您在编号列表中单击此图标，则该项目将转换为普通段落。

**插入/删除项目符号列表** - ![](images/BulletList_icon.svg)

在当前或下一个有效位置创建项目符号列表。 如果您在项目符号列表上单击此图标，则该项目将转换为普通段落。

**插入表** - ![](images/Table_icon.svg)

在当前或下一个有效位置插入表。 单击插入表图标以打开插入表对话框：

![](images/table-properties.png){width="550" align="left"}

您可以指定表中所需的行数和列数。 如果要将第一行保留为表标题，请选择“将第一行设置为标题”选项。 要向表中添加标题，请在标题字段中输入该标题。

插入表后，可以使用上下文菜单修改表。

![](images/table-context-menu_cs.png){width="550" align="left"}

使用表的上下文菜单，您可以：

- 插入单元格、行或列

- 在右方向和下方向合并单元格

- 水平或垂直拆分单元格

- 删除单元格、行或列

- 从表中创建代码片段

- 生成ID


您还可以在表的多个单元格、整行或列上定义属性。 例如，要对齐表单元格，请拖动并选择所需的单元格。 在“内容属性”面板\（位于右侧\）中，属性 **类型** 更改 **多条目**. 在“其他属性”部分中，选择 `@valign` 属性。 从值下拉列表中，选择要应用于选定表格单元格的所需文本对齐方式。

![](images/align-table-cell_cs.png){width="800" align="left"}

**插入图像** - ![](images/Image_icon.svg)

在当前或下一个有效位置插入图像。 单击插入图像图标以打开插入图像对话框，然后搜索并选择要插入的图像。

>[!NOTE]
>
> 您还可以通过将图像从本地系统拖放到文章中来添加图像。 在这种情况下，会使用 **上传资产** 工作流。  有关更多详细信息，请参阅 **上传资产** 工作流 [左面板](web-editor-features.md#id2051EA0M0HS) 中。


![](images/insert-image.png){width="650" align="left"}

您可以在“插入图像”对话框中为图像添加图像/图标标题和替换文本。

您可以通过在顶部的“要搜索的类型”栏中输入文件名来搜索所需的图像文件，还可以按路径\（要在\中搜索）、收藏集、文件类型和标记来筛选搜索结果。 找到所需的图像文件后，选择该文件，然后单击选择以在文档中插入该图像。 您可以插入各种格式的图像文件，例如 `.png`, `.svg`, `.gif`, `.jpg`, `.eps`, `.ai`, `.psd`，等等。

插入图像后，即可从“内容属性”面板更改高度、宽度、位置和属性。 单击图像文件，然后在右边栏的“内容属性”面板中进行更改。

![](images/image-properties.png){width="800" align="left"}

“源”字段显示插入图像文件的UUID。 通过将鼠标指针悬停在“源”字段上，可以找到插入图像文件的完整路径。 路径将显示在工具提示中。

您可以通过为图像文件提供“高度”或“宽度”值来调整图像大小。 自动保持图像的宽高比。 如果需要，您还可以通过单击锁定图标\（“维护宽高比”\的）并提供“高度”和“宽度”值来选择不维护图像文件的宽高比。

您还可以将图像的“放置”设置指定为“内嵌”或“中断”。 如果您选择使用“中断”放置选项，则可以选择在何处对齐图像\（左对齐、居中对齐或右对齐\）。

您还可以通过在 **属性** 字段。

>[!NOTE]
>
>您还可以在图像中定义可单击区域\（图像映射\）。 有关更多详细信息，请参阅 **插入/编辑图像映射** 中的功能描述 [左面板](web-editor-features.md#id2051EA0M0HS) 中。

**图像或媒体文件的上下文菜单**

您还可以使用上下文菜单对图像和媒体文件执行一些常见操作。 右键单击图像上的任意位置以调用上下文菜单。

上下文菜单提供用于剪切、复制或粘贴图像或媒体的选项。 您可以在选定元素之前或之后插入元素。 您还可以选择重命名或取消元素换行。 您可以在存储库中找到选定的图像或媒体，或在资产UI中查看文件预览。

上下文菜单中的其他选项允许您复制路径、编辑图像映射、创建代码片段或生成所选元素的ID。

**插入多媒体** - ![](images/insert-multimedia-icon.svg)

插入不同类型的多媒体文件。 单击“插入多媒体”图标，然后选择要插入的文件类型。 支持的多媒体格式包括：

- 音频文件
- 视频文件
- YouTube
- 维梅奥

选择音频或视频文件选项时，系统会向您显示存储库视图，以浏览并选择所需的文件。 如果选择YouTube或Vimeo，则会显示“插入多媒体”对话框。 将视频文件的链接粘贴到“Web链接”字段中，然后单击“插入”，以将视频添加到文档中的当前或下一个有效位置。

>[!NOTE]
>
> 添加YouTube视频链接时，您需要替换字符串 `watch?v=` with `embed` 中。 例如，要添加YouTube视频链接，请执行以下操作： `https://www.youtube.com/**watch?v**=WlIKQOrmZcs`，则需要将其添加为： `https://www.youtube.com/**embed/**WlIKQOrmZcs`. 此更改可确保将视频嵌入到AEM Site和PDF输出中。

**主题方案** -  ![](images/subject_scheme_panel-icon.svg)

主题方案映射是DITA映射的一种专门形式，用于定义分类主体和受控值。 根据您的要求，您可以创建主题方案映射并在根映射文件中引用它。 AEM指南允许您在主题方案中定义主题定义的嵌套级别层次结构。

您可以轻松地在主题方案映射中创建并使用主题方案。 将此映射添加为根映射后，主题方案即会显示在“主题方案”面板中。 “主题方案”面板以嵌套或分层方式显示可用的主题方案。

AEM指南还支持嵌套级别的主题方案映射，并且您可以在根主题方案映射下定义多个主题方案。

以下示例显示如何在AEM指南中使用主题方案。

1. 在您选择的工具中创建主题方案文件。 以下XML代码创建绑定 `platform` 属性。

   ```XML
   <?xml version="1.0" encoding="UTF-8"?>
   <!DOCTYPE subjectScheme PUBLIC "-//OASIS//DTD DITA Subject Scheme Map//EN" "subjectScheme.dtd">
   <subjectScheme id="GUID-4f942f63-9a20-4355-999f-eab7c6273270">
       <title>rw</title>
       <!-- Define new OS values that are merged with those in the unixOS scheme -->
       <subjectdef keys="os">
           <subjectdef keys="linux">    </subjectdef>
           <subjectdef keys="mswin">    </subjectdef>
           <subjectdef keys="zos">    </subjectdef>
       </subjectdef>
       <!-- Define application values -->
       <subjectdef keys="app" navtitle="Applications">
           <subjectdef keys="apacheserv">    </subjectdef>
           <subjectdef keys="mysql">    </subjectdef>
       </subjectdef>
       <!-- Define an enumeration of the platform attribute, equal to       each value in the OS subject. This makes the following values       valid for the platform attribute: linux, mswin, zos -->
       <enumerationdef>
           <attributedef name="platform">    </attributedef>
           <subjectdef keyref="os">    </subjectdef>
       </enumerationdef>
       <!-- Define an enumeration of the otherprops attribute, equal to       each value in the application subjects.       This makes the following values valid for the otherprops attribute:       apacheserv, mysql -->
       <enumerationdef>
           <attributedef name="otherprops">    </attributedef>
           <subjectdef keyref="app">    </subjectdef>
       </enumerationdef>
   </subjectScheme>
   ```

   ![](images/subject-scheme-panel.png){width="300" align="left"}

1. 保存扩展名为a.ditamap的文件，并将其上传到DAM中的任意文件夹。

   >[!NOTE]
   >
   > 您可以在父DITA映射中添加对主题方案文件的引用。

   ![](images/subject-scheme-root-map.png){width="550" align="left"}

1. 将父映射设置为 **用户首选项**. 将此映射添加为根映射后，主题方案即会显示在“主题方案”面板中。

   ![](images/subject-scheme-user-preferences.png){width="400" align="left"}

1. 在Web编辑器中，打开要在其中使用主题方案定义的文件。
1. 只需将所需的主题方案拖放到您的内容上，即可将主题方案应用到您的内容。 然后，内容将以定义的颜色突出显示。

## 右面板 {#id2051EB003YK}

右侧面板是一个永久性面板，其中包含有关当前选定文档的信息。

>[!NOTE]
>
> 右侧面板可调整大小。 要调整面板大小，请将光标置于面板边界上，光标将变为双向箭头，单击并拖动以调整面板宽度。

通过右侧面板，您可以访问以下功能：

**内容属性** -  ![](images/content-properties-icon.svg)

您可以通过单击右侧面板中的内容属性图标来访问内容属性功能。 “内容属性”面板包含有关文档中当前选定元素类型及其属性的信息。 您还可以通过从下拉列表中选择属性并指定属性的值来添加属性。

>[!NOTE]
>
> 即使您的主题包含引用的内容，您也可以使用“属性”面板在其中添加属性。

如果您的管理员为属性创建了配置文件，则您将获得这些属性及其配置的值。 使用“内容属性”面板，您可以选择这些属性，并将其分配给主题中的相关内容。 这样，您还可以创建条件内容，然后使用该内容创建条件输出。 有关使用条件预设生成输出的更多信息，请参阅 [使用条件预设](generate-output-use-condition-presets.md#).

![](images/properties-tab-attributes_cs.png){width="300" align="left"}

**文件属性** -  ![](images/topic-properties-icon.svg)

单击右侧面板中的“文件属性”图标，查看所选文件的属性。 “文件属性”包含以下两个部分：

**常规**

在常规部分，您可以访问以下功能：

![](images/file-properties-general.png){width="300" align="left"}

- **名称**:显示所选主题的文件名。 文件名会以超链接方式链接到选定文件的属性页面。
- **ID**:显示所选主题的ID。
- **元数据标记**:这些是主题的元数据标记。 这些值是从属性页面的标记字段中设置的。
- **语言**:显示主题的语言。 它是从属性页面的语言字段中设置的。
- **创建于**:显示创建主题的日期和时间。
- **签出者**:显示已签出主题的用户。
- **文档状态**:您可以选择并更新当前打开的主题的文档状态。 有关更多详细信息，请参阅 [文档状态&#x200B;](web-editor-document-states.md#)*.*

**注意：** 您可以将“文件”属性中各个字段的属性值复制到剪贴板。

**引用**

“引用”(References)部分允许您访问以下功能：

![](images/file-properties-references.png){width="300" align="left"}

- **在中使用**:“在中使用”(Used In)引用列出了引用或使用当前文件的文档。
- **传出链接：** “传出链接”列出了当前文档中引用的文档。

将鼠标悬停在文件引用上，并在工具提示中获取文件的文件路径和UUID。

**注意：** “Used In（在中使用）”和“Outgoin（传出）”引用均超链接到文档。 您可以轻松打开和编辑链接的文档。

除了打开文件之外，您还可以使用 **选项** 菜单。 您可以执行的一些操作包括“编辑”、“预览”、“复制UUID”、“复制路径”、“添加到收藏夹”、“属性”和“打开映射”功能板。

**审阅** -  ![](images/review-icon.svg)

单击审阅图标可打开审阅面板，您可以在其中为当前打开的文档创建审阅任务。

![](images/review-panel-before-opening.png){width="300" align="left"}

如果已创建多个审核项目，则可以从下拉列表中选择一个并访问审核注释。

使用审阅面板，可以查看和发布对针对该主题提供的评论的回复。 您可以逐个接受或拒绝注释。

有关更多信息，请参阅 [地址审阅注释](review-address-review-comments.md#).

**跟踪的更改** -  ![](images/track-change-icon.svg)

使用右侧面板的“跟踪的更改”功能，可以查看文档中所有更新的信息。 您还可以搜索对文档进行的任何特定更新。

>[!NOTE]
>
> “跟踪的更改”功能显示已使用主工具栏的“启用/禁用跟踪更改”功能跟踪的所有更新。 有关更多详细信息，请参阅 [启用/禁用跟踪更改](#id205DF0203Y4).
