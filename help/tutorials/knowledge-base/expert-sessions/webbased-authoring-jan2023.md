# 创作基于Web的内容

AEM指南附带一个用于创作结构化内容的内置Web编辑器。 您可以使用基于Web的简化创作体验，但可以选择使用或启用诸如条件内容验证、拼写检查、内容重用、引用验证、友好名称和创作数学公式等强大功能。

## 会话概述

在本课程中，您将学习如何使用Web编辑器创建、发布和管理内容。

- 使用Web编辑器功能提高作者的工作效率
- 扩展和集成可用功能
- 通过自定义样式或品牌策略来个性化Web编辑器
