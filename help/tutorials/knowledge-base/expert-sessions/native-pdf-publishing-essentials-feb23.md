# 本机PDF发布要点

AEM指南具有内置的PDF发布引擎，该引擎为用户提供创建、开发和发布PDF格式的内容的功能。 其丰富的功能集允许创建各种页面布局、自定义样式表以及设计无缝内容发布所需的各种PDF组件。

## 会话概述

在本课程中，您将学习如何在AEM指南中使用本机pdf引擎发布PDF
